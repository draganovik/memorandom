## Installation

Install dependencies with npm

```bash
$ npm install
```

## Run Locally

Serve with hot reload at localhost:3000

```bash
$ npm run dev
```

## Deployment

Generate static project files (in _docs_ folder)

```bash
$ npm run build
```
